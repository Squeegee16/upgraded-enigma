WSJT-X Setup

Configure WSJT-X to send UDP data:

  1. Open WSJT-X
  2. Go to File → Settings
  3. Click Reporting tab
  4. Set UDP Server: localhost
  5. Set Port: 2237
  6. Check Accept UDP requests
  7. Click OK
